## Prerequisites

* Python 3.8
  * How to install latest stable release [here](https://tecadmin.net/install-python-3-8-ubuntu/)
* Scrapy 2.0.1
  * `pip install -r requirements.txt`

## Getting Started

```bash
# Check scrapy.org
scrapy runspider myspider.py
```
